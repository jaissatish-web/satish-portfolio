Loop Check Checklist

📅 Jan 2025🏷️ Commissioning, QA/QC

The Hook

Control valve refused to respond at NSRP startup. Loop check revealed reverse AO card configuration — caught on paper before energizing.

Core Concept

  • Signal Verification: 4-20mA loop check with 250Ω resistor
  • Alarm Check: Verify LL, L, H, HH setpoints in DCS
  • Valve Stroke: Manual 0-100% with feedback verification
  • Configuration: Check tag numbers, engineering units, action
  • P&ID Compliance: Cross-verify against loop diagrams

Interactive Simulator

Real-World Story

ADNOC fired heater lost fuel gas during commissioning. Investigation showed control valve XV-201 AO card was configured for reverse action. Configuration corrected before automatic mode entry.

Takeaway: Loop checks catch configuration errors before startup.
